🛒 E-Commerce Web Application (MERN Stack)

 📌 Overview

This project is a full-stack E-Commerce web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It allows users to browse products, manage a cart, and place orders through an interactive and responsive interface.


🚀 Features

* 🛍️ Product Listing & Product Details Page
* 🛒 Add to Cart / Remove from Cart
* 📦 Order Placement & Summary
* 🔎 Search & Filter Products
* 📱 Fully Responsive UI
* ⚡ Fast API-based data fetching

🏗️ Tech Stack

### Frontend

* React.js
* CSS
* HTML5

### Backend

* Node.js
* Express.js
* mongoD


⚙️ Installation & Setup

### 1. Clone the repository

### 2. Install dependencies

### 3. Setup Environment Variables

### 4. Run the application

#### Start Frontend: npm run dev
